# ✅ Smoke Test сценарий (ручной)

## Preconditions
- `TELEGRAM_BOT_TOKEN` задан
- `KIE_API_KEY` задан
- `DATABASE_URL` задан
- (опционально) `ADMIN_ID` задан

## Steps
1. `/start` → убедиться, что показывается главное меню.
2. Выбрать категорию → выбрать модель → ввести параметры → подтвердить.
3. Убедиться, что появляется сообщение “job created / генерация началась”.
4. Нажать “cancel” → убедиться, что происходит возврат/отмена и статус понятен.
5. Отправить фото/док/аудио в режиме ожидания текста → получить понятную подсказку.
6. `/diag` (если `ADMIN_ID` задан) → убедиться, что отображаются BOT_MODE/webhookInfo/lock_status/instance_id/last_start_time.
