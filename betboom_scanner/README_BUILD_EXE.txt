   ИНСТРУКЦИЯ ПО СБОРКЕ EXE-ФАЙЛА
═══════════════════════════════════════════════════════════════

ВАЖНО: EXE требует установленный Python для сборки!
После сборки exe работает БЕЗ Python.

ШАГ 1: Установите Python (если не установлен)
───────────────────────────────────────────────
1. Скачайте с https://www.python.org/downloads/
2. При установке ОБЯЗАТЕЛЬНО поставьте галочку "Add Python to PATH"
3. Перезагрузите компьютер

ШАГ 2: Сборка EXE
─────────────────
1. Откройте папку betboom_scanner
2. Дважды кликните на файл: build_exe.bat
3. Дождитесь завершения (3-5 минут)
4. EXE будет в папке: dist\BB_Scanner.exe

ШАГ 3: Использование
─────────────────────
1. Скопируйте dist\BB_Scanner.exe на рабочий стол
2. Запустите его - больше ничего не нужно!

ПРИМЕЧАНИЯ:
───────────
- EXE файл будет около 50-100 МБ (включает Python + Playwright + Chromium)
- При первом запуске exe может быть медленным (распаковка)
- Все данные сохраняются в папке app_data рядом с exe

ПРОБЛЕМЫ ПРИ СБОРКЕ?
────────────────────
1. Если ошибка "pyinstaller не найден":
   - Откройте командную строку
   - Введите: pip install pyinstaller
   - Запустите build_exe.bat снова

2. Если ошибка с playwright:
   - Убедитесь, что установлен Chromium:
     python -m playwright install chromium

3. Если exe не запускается:
   - Попробуйте собрать с консолью (для отладки):
     Измените в build_exe.bat: --windowed на --console
