📋 ЧТО НУЖНО:
  ✓ Windows 10/11
  ✓ Интернет-соединение
  ✓ ~200 МБ свободного места

🚀 КАК ЗАПУСТИТЬ:
  1. Дважды кликните на: start_scanner.bat
  2. При первом запуске подождите ~2-3 минуты (установка зависимостей)
  3. Откроется окно сканера
  4. Нажмите "Start" для начала сканирования

🎯 ЧТО ДЕЛАЕТ СКАНЕР:
  • Сканирует live-настольный теннис на BetBoom
  • Ищет сигналы по стратегии:
    - Фаворит матча (кф ≤ 1.20)
    - Ведёт 2:0 по сетам
    - 3-й сет только начался (0:0, 1:0, 0:1)
    - На 3-й сет кф ≥ 1.90

🔊 УВЕДОМЛЕНИЯ:
  • Звуковой сигнал
  • Windows toast уведомление
  • Запись в signals.csv и signals.sqlite

📁 ФАЙЛЫ:
  • signals.csv - все сигналы в CSV формате
  • signals.sqlite - база данных с сигналами
  • errors.log - логи ошибок
  Все файлы в папке: app_data/

❓ ПРОБЛЕМЫ:
  • Смотрите README.md для подробной документации
  • Проверьте app_data/errors.log при ошибках
