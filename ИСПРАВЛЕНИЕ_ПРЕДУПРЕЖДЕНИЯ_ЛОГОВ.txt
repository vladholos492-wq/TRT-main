   БЫЛО:
   ```python
   if not history:
       logger.warning(f"History file {GENERATIONS_HISTORY_FILE} is empty")
       return []
   ```

   СТАЛО:
   ```python
   if not history:

       return []
   ```

   ОБОСНОВАНИЕ:
   - Пустой файл истории - это нормальная ситуация
   - Не требует внимания администратора
   - INFO уровень более подходящий для информационных сообщений
   - Добавлен комментарий для ясности
