# Быстрое исправление ошибок импорта

## Проблема
VS Code показывает ошибки импорта (красные подчеркивания) для модулей Python.

## Решение за 3 шага

### Шаг 1: Установите Python (если еще не установлен)
1. Скачайте Python: https://www.python.org/downloads/
2. При установке **обязательно** отметьте галочку **"Add Python to PATH"**
3. Перезапустите компьютер после установки

### Шаг 2: Установите зависимости
Откройте терминал в VS Code (`Ctrl + ~`) и выполните:

```powershell
powershell -ExecutionPolicy Bypass -File setup_python.ps1
```

Или вручную:
```bash
python -m pip install -r requirements.txt
```

### Шаг 3: Перезапустите VS Code
1. Нажмите `Ctrl + Shift + P`
2. Введите: `Developer: Reload Window`
3. Нажмите Enter

## Если ошибки остались

1. **Выберите интерпретатор Python в VS Code:**
   - `Ctrl + Shift + P`
   - `Python: Select Interpreter`
   - Выберите Python 3.8 или выше

2. **Проверьте, что зависимости установлены:**
   ```bash
   python -c "import telegram; print('OK')"
   ```

3. **Если Python не найден в терминале:**
   - Перезагрузите компьютер
   - Или добавьте Python в PATH вручную через "Переменные среды"

## Альтернативный способ

Если ничего не помогает, используйте готовый скрипт установки:

```bash
setup.bat
```

Этот скрипт автоматически установит все зависимости и создаст файл `.env`.
