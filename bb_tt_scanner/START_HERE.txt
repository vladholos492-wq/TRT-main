🚀 БЫСТРЫЙ СТАРТ (2 минуты)

1. ЕСЛИ У ВАС ЕСТЬ .exe:
   → Запустите dist\BB_TT_Scanner.exe
   → Готово! Всё работает автоматически

2. ЕСЛИ НУЖНО СОБРАТЬ ИЗ ИСХОДНИКОВ:
   → Установите Python 3.11+
   → Выполните: pip install -r requirements.txt
   → Выполните: python -m playwright install chromium
   → Запустите: python run.py

3. ПЕРВЫЙ ЗАПУСК:
   → Откроется браузер
   → Войдите в BetBoom (1 раз)
   → После входа сканирование начнётся автоматически
   → Сессия сохранится (больше логиниться не нужно)
