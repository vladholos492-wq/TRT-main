========================================
  КАК ЗАПУСТИТЬ БОТА
========================================

БЫСТРЫЙ ЗАПУСК:
---------------
1. Дважды кликните на файл: run_bot_direct.bat
2. Бот запустится и будет работать
3. Для остановки нажмите Ctrl+C

АЛЬТЕРНАТИВНЫЙ ЗАПУСК:
---------------------
1. Откройте PowerShell или CMD
2. Перейдите в эту папку:
   cd C:\Users\user\Desktop\12-main
3. Запустите:
   python run_bot.py

ПРОВЕРКА РАБОТЫ:
----------------
1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте команду: /start
4. Бот должен ответить приветствием

ДОСТУПНЫЕ КОМАНДЫ:
------------------
/start - Начать работу с ботом
/help - Показать список команд
/search [запрос] - Поиск в базе знаний
/ask [вопрос] - Задать вопрос
/add [знание] - Добавить новое знание

ВАЖНО:
------
- Убедитесь, что Python установлен (python --version)
- Файл .env должен содержать ваш токен бота
- Бот должен быть запущен, чтобы отвечать на сообщения
