📦 АВТОМАТИЧЕСКАЯ УСТАНОВКА (РЕКОМЕНДУЕТСЯ)
═══════════════════════════════════════════════════════════════

1. Убедитесь, что установлен Python 3.8 или выше
   Скачать: https://www.python.org/downloads/
   ⚠️ ВАЖНО: При установке отметьте "Add Python to PATH"

2. Дважды кликните на файл: setup.bat

3. Следуйте инструкциям на экране:
   - Скрипт установит все зависимости
   - Запросит необходимые данные
   - Создаст файл .env

4. Запустите бота: run_bot_simple.bat

═══════════════════════════════════════════════════════════════

📋 ЧТО НУЖНО ПОДГОТОВИТЬ
═══════════════════════════════════════════════════════════════

1. TELEGRAM_BOT_TOKEN
   - Откройте @BotFather в Telegram
   - Отправьте /newbot
   - Создайте бота и скопируйте токен

2. KIE_API_KEY
   - Зарегистрируйтесь на https://api.kie.ai
   - Получите API ключ в личном кабинете

3. ADMIN_ID (опционально)
   - Ваш Telegram User ID
   - Можно узнать у @userinfobot
   - По умолчанию: 6913446846

═══════════════════════════════════════════════════════════════

🔧 РУЧНАЯ УСТАНОВКА
═══════════════════════════════════════════════════════════════

Если автоматическая установка не работает:

1. Установите зависимости:
   python -m pip install -r requirements.txt

2. Создайте файл .env (скопируйте из env_template.txt)
   и заполните все необходимые данные

3. Запустите бота:
   python run_bot.py
